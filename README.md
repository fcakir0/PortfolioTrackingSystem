# Portfolio Tracking System

Bu proje, Java Swing kullanılarak geliştirilmiş bir portföy takip uygulamasıdır.
Uygulama, kullanıcıların yatırım portföylerini oluşturabilmesini, alım-satım
işlemlerini kaydedebilmesini ve portföy performansını takip edebilmesini
amaçlamaktadır.

---

## Projenin Amacı
Bu projenin amacı, kullanıcıların farklı piyasalarda yer alan finansal varlıklarını tek
bir platform üzerinden takip edebilmesini sağlamak ve portföy durumunu anlaşılır bir şekilde
sunmaktır.

---
## Kullanılan Teknolojiler
- Java
- Java Swing (Masaüstü arayüz)
- PostgreSQL (Veritabanı)
- JDBC
- Maven

---

## Uygulama Özellikleri
- Kullanıcı girişi
- Portföy oluşturma ve yönetme
- Alım-satım işlemlerinin kaydedilmesi
- Ortalama maliyet hesaplama
- Kar/Zarar hesaplama
- Portföy değerinin görüntülenmesi
- Açık / koyu tema desteği
- Dinamik ve kullanıcı dostu arayüz

---

## Proje Yapısı
Proje, katmanlı mimari prensiplerine uygun şekilde geliştirilmiştir.

- `ui` : Java Swing arayüz bileşenleri
- `service` : İş mantığı
- `repository` : Veritabanı işlemleri
- `model` : Varlık (Entity) sınıfları
- `util` : Yardımcı sınıflar

---

## Kurulum ve Çalıştırma
1. PostgreSQL üzerinde gerekli veritabanı ve tabloları oluşturun
2. Veritabanı bağlantı bilgilerini proje içerisinden yapılandırın
3. Projeyi IntelliJ IDEA veya benzeri bir Java IDE ile açın
4. Ana uygulama sınıfını çalıştırın

---

- Kalıtım (Inheritance)
- Çok biçimlilik (Polymorphism)
